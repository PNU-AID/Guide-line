# 공지사항

- 디스코드 서버에서 이름을 모두 본인 이름으로 변경 부탁드립니다.
- 스터디 / 프로젝트는 자유롭게 만드실 수 있고 팀장을 정하고 운영진에게 알려주면 디스코드 역할 및 채널을 생성해드립니다.
- 질문은 질문 채널에서 자유롭게 부탁드립니다.
- 모든 스터디 회의는 팀장에게 문의 후 자유롭게 청강 가능합니다.
- 신입 부원은 반드시 스터디 / 프로젝트 / 대회를 진행하셔야 합니다.
- 활동중인 부원은 학기 마지막 세미나는 반드시 참여하셔야 합니다.
- 폭력적, 차별적인 언어는 금지합니다.
- 커뮤니케이션 예절을 지켜주세요.

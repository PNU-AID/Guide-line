질문은 자유롭게 하면 됩니다.
Please feel free to ask any questions.

답변이 없다고 주눅들지마세요. 질문을 하는 과정 그 자체에서 성장하는 것입니다.
If it's too difficult or everyone is busy, there might not be an answer, but don't be discouraged! Asking questions is a way to grow.

## Question Format

1. 어디서 이런 문제가 나왔는지? (Where did the problem come from?)
2. 현재 궁금한 문제 혹은 해결하지 못한 문제가 무엇인지 디테일하게 설명 (Explain in detail what problems you are currently curious about or have not been able to solve.)
3. 틀린 답이더라도 본인이 생각한 해결법 혹은 찾은 방법 (Even if the answer is wrong, the solution you came up with or the method you found)


**Example**

- [ ] 현재 운영체제(current OS): Window11
- [ ] 현재 사용언어 및 버전(programming language and its version): Python 3.9.13
- [ ] 현재 가상환경의 라이브러리 버전(version of libraries in virtual enviornment): 
- [ ] 실행한 명령어(Commnad what you run):
- [ ] 해결하고 싶은 문제 혹은 오류(Error What you want to solve):
- [ ] 문제가 생긴 코드(Code that casued the problem):
- [ ] 본인이 생각한 해결방법(The solution you came up with):
- [ ] 찾아본 사이트 링크(Link to the site you visited):

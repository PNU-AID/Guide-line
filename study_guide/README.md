# 스터디 가이드

_모집글은 디스코드 스터디 모집글 채널에 올려주시면 됩니다._
## 모집글 형식

예시
> **스터디 내용 : 고급 파이썬 스터디**
> **모집 인원 : 4명**
> **대면/비대면 여부 : 비대면(월 1회 대면)**
> **모집 대상 : 매일 1시간 이상 개인 공부가능하신분, 파이썬으로 프로그래밍 개발 경험**
> (아래 항목은 선택사항)
> **공부 내용 : 파이썬 코드 최적화 및 프로파일링**

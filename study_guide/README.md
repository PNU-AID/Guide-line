# 스터디 가이드

_모집글은 디스코드 스터디 모집글 채널에 올려주시면 됩니다._
## 모집글 형식

예시
> **스터디 내용 : 고급 파이썬 스터디**<br/>
> **모집 인원 : 4명**<br/>
> **대면/비대면 여부 : 비대면(월 1회 대면)**<br/>
> **모집 대상 : 매일 1시간 이상 개인 공부가능하신분, 파이썬으로 프로그래밍 개발 경험**<br/>
> (아래 항목은 선택사항)<br/>
> **공부 내용 : 파이썬 코드 최적화 및 프로파일링**<br/>

## 🚨 스터디 / 프로젝트 필수사항

- 누구든지 청강가능 하도록 해야합니다
- 신입 부원인 경우 한 학기 이상 스터디 / 프로젝트 / 대회 참여할 것
- 스터디 / 프로젝트 기간은 새로운 스터디나 프로젝트를 위해 최대한 학기 끝 혹은 방학 끝으로 맞출것(대회는 예외)
- 스터디에서 공부한 내용 블로그, 깃허브, 노션중 하나로 기록하고 공유하고, 스터디의 시작과 종료시 채널 개폐를 위해 스터디장이 운영진에게 공유부탁드립니다

## Question Format

1. 어디서 이런 문제가 나왔는지
2. 현재 궁금한 문제 혹은 해결하지 못한 문제가 무엇인지 디테일하게 설명
3. 본인이 생각한 해결법 혹은 찾은 방법


**Example**
- [ ] 현재 운영체제: Window11
- [ ] 현재 사용언어 및 버전: Python 3.9.13
- [ ] 현재 가상환경의 라이브러리 버전: 
- [ ] 실행한 명령어:
- [ ] 해결하고 싶은 문제 혹은 오류:
- [ ] 문제가 생긴 코드:
- [ ] 본인이 생각한 해결방법:
- [ ] 찾아본 사이트 링크:
